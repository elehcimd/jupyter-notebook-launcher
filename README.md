## Know your shortcuts
Action                 | Shortcut
-----------------------| -----------------------
Open server in browser | <kbd>Click</kbd> on server
Shut down server       | <kbd>Alt</kbd> <kbd>Click</kbd> on server
New server             | <kbd>Cmd</kbd> <kbd>N</kbd>
Quit                   | <kbd>Cmd</kbd> <kbd>Q</kbd>
